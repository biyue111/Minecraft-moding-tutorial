## 重要命令

`git config --global user.email useremail`

`git config --global user.name username`

用自己的email和名称替换命令最后useremail和username

`git clone` 克隆到本地

`git add` 将文件加入到仓库记录中

`git status` 查看状态

`git commit` 
 
`git commit -a -m ""` 

`git push` 将本地修改“推”到远程仓库中

`git pull` 将远程仓库中的修改“拉”到本地仓库
